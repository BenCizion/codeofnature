자주 묻는 질문 및 정책

한도를 초과하면 어떻게 되나요?

무료 플랜은 서비스가 일시 중지되며 광고가 유지됩니다. 유료 플랜은 3개월 연속 초과 시 상위 플랜으로 전환을 제안드리며, 즉각적인 차단은 하지 않습니다.

이미지 댓글 차감 기준은요?

시스템 부하를 고려하여 이미지 댓글 1개 작성 시, 필터링 수 3건이 차감됩니다. (텍스트 댓글은 1건 차감)

API 비용은 별도인가요?

네, API 사용은 C(Business) 등급 이상부터 옵션으로 추가 가능하며, 호출량은 기본 필터링 수에 합산되어 차감됩니다.

담당자 배정은 언제 되나요?

전담 기술 지원 매니저는 F(Enterprise) 등급 고객에게만 배정됩니다. 그 외 등급은 이메일 및 커뮤니티 지원을 원칙으로 합니다.
