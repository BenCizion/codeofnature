RECOMMENDED (주요 언론사 적용)

A. 자체 회원 연동 수정

Fast Track

기존 회원 시스템의 로그인 세션(Cookie/LocalStorage) 정보를 라이브리 Vault와 안전하게 공유하는 방식입니다.

  • 빠른 도입: 개발 공수를 획기적으로 단축
  • 레거시 호환: 기존 구조 변경 불필요
  • 검증된 보안: 주요 언론사 도입으로 입증

localStorage.setItem('livere_auth', token);

// 프론트엔드 작업만으로 연동 가능

STANDARD

B. OIDC 표준 프로토콜

Global Standard

업계 표준인 OpenID Connect 인증 프로토콜을 통해 인증 정보를 서버 간 교환하는 방식입니다.

  • 표준 준수: OAuth 2.0 / OIDC 기반 인증
  • 높은 확장성: 타 시스템 연동성 확보
  • 자체 인증 서버(OP) 구축 필요 (높은 공수)

GET /authorize?response_type=code

// 표준 인증 서버 구축 필요
