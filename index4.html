유연한 기술 구현 방식

고객사의 개발 환경과 니즈에 맞춰 최적의 연동 방식을 지원합니다.

EASY TO ADAPT (빠른 적용)

A. 자체 회원 연동 수정

Fast Track

기존 회원 시스템의 로그인 세션(Cookie/LocalStorage) 정보를 라이브리 Vault와 안전하게 공유합니다.

  • 빠른 도입: 개발 공수 획기적 단축
  • 레거시 호환: 기존 구조 변경 불필요
localStorage.setItem('livere_auth', token);
STANDARD (추천하는 방식)

B. OIDC 표준 프로토콜

Global Standard

업계 표준인 OpenID Connect 인증 프로토콜을 통해 인증 정보를 서버 간 교환하는 방식입니다.

  • 표준 준수: OAuth 2.0 / OIDC 기반
  • 높은 확장성: 타 시스템 연동성 확보
GET /authorize?response_type=code
